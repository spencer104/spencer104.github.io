Dashbaord Project

For this project, I created a Power BI dashboard to visualize the Canadian rental housing market in June 2024 using prices from RentFaster.ca. The data was acquired from Kaggle and imported using a CSV file. The provided data was cleaned using the Power Query function in Power BI, where I removed duplicates, converted data types, and set up new columns. However, some manual work and script creation were required to clean the square footage category due to inconsistencies in how each user described the square footage of the rental property. The dashboard allows users to manipulate the data based on price, price per square footage, price per bed, and provinces.Below are the links to the Kaggle dataset, a photo of the dashboard (as I currently do not have a workspace to host it), and the GitHub repository where you can download the dashboard.
